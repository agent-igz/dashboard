# Dashboard UI

## Launcher fields
- topic / opportunity space
- number of ideas
- mode: simple / full / custom
- rubric preset
- optimize-for target
- constraints
- desired business model
- agent role expectation: core-engine / service-layer / internal-advantage

## Results view
- ranked shortlist
- killed ideas and reasons
- per-role structured outputs
- weighted score breakdown
- orchestrator synthesis
- rerun actions

## Rerun actions
- rerun whole workflow
- rerun one role/stage
- duplicate run with modified inputs
